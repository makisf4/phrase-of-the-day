Πολιτική Απορρήτου

Γενικές Πληροφορίες

Η παρούσα Πολιτική Απορρήτου περιγράφει τον τρόπο με τον οποίο η ιστοσελίδα "Η Φράση της Ημέρας που Σε Περιγράφει" συλλέγει, χρησιμοποιεί και προστατεύει τα δεδομένα των επισκεπτών της.

Αυτή η ιστοσελίδα είναι ένα στατικό site χωρίς backend, χωρίς λογαριασμούς χρηστών και χωρίς φόρμες. Δεν συλλέγουμε προσωπικά δεδομένα μέσω φορμών ή καταγραφών.

Ποια δεδομένα συλλέγονται

Δεν συλλέγουμε προσωπικά δεδομένα. Η ιστοσελίδα λειτουργεί ως static site και δεν διαθέτει backend server, βάση δεδομένων ή σύστημα αποθήκευσης δεδομένων.

Το μόνο που μπορεί να αποθηκευτεί τοπικά στη συσκευή σου είναι μέσω του localStorage του browser (για παράδειγμα, για την παρακολούθηση συνεχόμενων επισκέψεων). Αυτά τα δεδομένα δεν αποστέλλονται σε server και παραμένουν αποκλειστικά στη συσκευή σου.

Ο server hosting (Vercel) μπορεί να καταγράφει τυπικά server logs (IP addresses, timestamps) για τεχνικούς σκοπούς, σύμφωνα με την πολιτική απορρήτου του hosting provider.

Cookies & Διαφημίσεις

Η ιστοσελίδα μπορεί να εμφανίζει διαφημίσεις μέσω της Google AdSense. Η Google και οι συνεργάτες της μπορεί να χρησιμοποιούν cookies ή άλλους αναγνωριστικούς (identifiers) για τους εξής σκοπούς:

  • Παροχή διαφημίσεων (προσωποποιημένων ή μη, ανάλογα με τη συναίνεσή σου)
  • Μέτρηση και αναφορά αποτελεσμάτων
  • Πρόληψη απάτης
  • Περιορισμός συχνότητας εμφάνισης διαφημίσεων

Συναίνεση (EEA): Αν βρίσκεσαι στην Ευρωπαϊκή Οικονομική Ζώνη, θα σου ζητηθεί συναίνεση πριν από την παροχή προσωποποιημένων διαφημίσεων. Μπορείς να αποδεχτείς ή να απορρίψεις τη συναίνεση, και να αλλάξεις τις επιλογές σου ανά πάσα στιγμή μέσω του link "Ρυθμίσεις cookies" στον πίνακα συνδέσμων (footer) της σελίδας.

Έλεγχος διαφημίσεων: Μπορείς να διαχειριστείς τις προτιμήσεις διαφημίσεων μέσω των Ρυθμίσεων Διαφημίσεων της Google. Επίσης, μπορείς να αποσύρεις ή να αλλάξεις τη συναίνεσή σου οποιαδήποτε στιγμή μέσω του CMP (Consent Management Platform) που χρησιμοποιεί η ιστοσελίδα.

Μπορείς επίσης να διαχειριστείς τα cookies μέσω των ρυθμίσεων του browser σου.

Τρίτες Υπηρεσίες

Η ιστοσελίδα φιλοξενείται στο Vercel. Για πληροφορίες σχετικά με την πολιτική απορρήτου του Vercel, μπορείς να επισκεφτείς την επίσημη ιστοσελίδα τους.

Αν προστεθούν διαφημίσεις στο μέλλον, η Google AdSense μπορεί να συλλέγει δεδομένα σύμφωνα με τη δική της πολιτική απορρήτου. Σε αυτή την περίπτωση, θα ενημερωθείς μέσω της παρούσας πολιτικής απορρήτου.

Επικοινωνία

Για οποιαδήποτε ερώτηση σχετικά με την πολιτική απορρήτου, μπορείς να επικοινωνήσεις μέσω του GitHub repository αυτού του project ή μέσω των καναλιών επικοινωνίας που μπορεί να διαθέτει ο δημιουργός της ιστοσελίδας.
